## Prerequisites

Before you begin, make sure the following tools are installed on your machine:

- **Python 3.10 or higher** — required for the FastAPI backend and ML pipeline
- **Node.js 18 or higher** — required for the React/TypeScript frontend
- **git** — for cloning the repository

<Tip>
  Run `python --version`, `node --version`, and `git --version` to verify your installed versions before proceeding.
</Tip>

## Repository structure

HealthIA is a full-stack application composed of two independent services that run separately:

| Directory | Service | Language / Framework |
|-----------|---------|----------------------|
| `HEALTHIA/` | Backend API + ML pipeline | Python, FastAPI, scikit-learn |
| `frontend/` | Web interface | TypeScript, React, Vite, Tailwind CSS |

<Note>
  The backend and frontend are separate services with their own dependency manifests and development servers. You need to start both independently to run the full application locally. See [Backend development](/development/backend) and [Frontend development](/development/frontend) for setup instructions for each service.
</Note>
